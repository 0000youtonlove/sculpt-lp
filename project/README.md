## 🚀 GitHub Pages へのデプロイ手順

1. このリポジトリを `Public` で作成
2. Settings → Pages → Branch: `main` / `/ (root)` を選択して Save
3. 数分後に `https://yourusername.github.io/リポジトリ名/` で公開される
4. `index.html` の `og:url` と `canonical` タグのURLを自分のURLに更新する
